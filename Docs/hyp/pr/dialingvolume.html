Properties

dialingVolume

Supported By

  • OpenXION (1.2, 1.3, 1.4)
  • OpenXION Audio Module (1.2, 1.3, 1.4)
  • HyperTalk (2.2)

Applies To

Global environment

Description

The dialingVolume property determines the volume of the dial tones produced by the dial command. The default value is 7.

Note

In HyperTalk, as in old versions of Mac OS Classic, the volume was restricted to an integer between 0 and 7. OpenXION uses the same range for compatibility, but allows any number in order to give finer control over volume.
