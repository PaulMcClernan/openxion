Functions

sound

Supported By

  • OpenXION (1.2, 1.3, 1.4)
  • OpenXION Audio Module (1.2, 1.3, 1.4)
  • HyperTalk (2.2)

Syntax

the sound

sound ()

Examples

put the sound

wait until the sound is "done"

Description

The sound function returns the name of the instrument currently playing on the current soundChannel, or the string "done" if no instrument is currently playing.

Note

The "done" string is a literal; it is not a constant like zero or true. In HyperTalk code, the "done" literal is often left unquoted, but in XION this is highly discouraged.
