Other Keywords

in

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Base (1.0, 1.1, 1.2, 1.3, 1.4)
  • HyperTalk (2.2)

Syntax

[the] chunk in parent

[the] descriptor in parent

[the] function-name in argument

[the] property-name in object

Description

The in keyword is used in several places to describe child-parent relationships between objects and arguments to function calls.

It is used in chunk expressions: char 32 in steve

It is used in descriptors for child objects: file "doc.xnd" in folder "Documentation"

It is used in function calls: the sqrt in sixteen

It is used in property descriptors: the location in myRectangle

It is used in operators: steve is an element in bill

It is used in certain control structures: repeat for each myElement in myList

Note

The in and of keywords are synonyms practically everywhere in both XION and HyperTalk, even when it would otherwise make no sense (e.g., the sqrt in 4 or "e" is of "hello").

Synonym

of
