Constants

down

Supported By

  • HyperTalk (2.2)

Example

wait until the mouse is down

Description

A string consisting of the text "down" (without quotes).

Note

The down constant is a rare example of HyperTalk defining something other than a command or function that XION does not. It was decided that since left and right cannot be defined as constants alongside up and down (as they conflict with the left and right functions), then up and down should not be defined as constants either. Furthermore, the up and down constants are used in an environment-specific context for describing the state of mouse buttons and keyboard keys, which does not apply to XION.
