Data Types

point

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Standard Module (1.0, 1.1, 1.2, 1.3, 1.4)
  • HyperTalk (2.2)

Description

The point data type represents a point in a Cartesian coordinate system with integer coordinates.

Notes

The string representation of a point is the string representation of the X coordinate, followed by a comma, followed by the string representation of the Y coordinate.

The string representation of a point always includes a comma; it does not depend on the setting of the itemDelimiter property.

Compatibility

In HyperTalk, you can test for a point with the is an operator, but the test is broken: if the value is not a comma-delimited list of integers, the expression will trigger an "Expected integer here" script error instead of evaluating to false.
