Commands

get

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Standard Module (1.0, 1.1, 1.2, 1.3, 1.4)
  • HyperTalk (2.2)

Syntax

get expression

Expression yields any value.

Examples

get the short name of file 1

get 2+3

get the date

Description

The get command puts the value of any expression into the local variable it. That is, get expression is the same as put expression into it.
