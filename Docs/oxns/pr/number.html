Properties

number

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Standard Module (1.0, 1.1, 1.2, 1.3, 1.4)
  • HyperTalk (2.2)

Applies To

  • Chunk types
  • Lists
  • User-defined objects

Examples

get the number of elements in steve

get the number of words in bill

get the number of andy

Description

For chunk types, the number property determines the number of chunks in a range.

For lists, the number property determines the number of items in a list.

For objects, the number property determines the index of an object.
