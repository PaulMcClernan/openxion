Properties

rowDelimiter

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Standard Module (1.0, 1.1, 1.2, 1.3, 1.4)

Applies To

Global environment

Example

set rowDelimiter to backspace

Description

The rowDelimiter property determines or changes the character that delimits row chunks. The default value is Unicode character U+FFF1.

See Also

row, itemDelimiter, columnDelimiter
