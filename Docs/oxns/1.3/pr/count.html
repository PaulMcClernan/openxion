Properties

count

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Standard Module (1.0, 1.1, 1.2, 1.3, 1.4)

Applies To

Directory-type file system objects (disks and folders)

Example

get the count of the first volume

Description

The count property determines the number of files and folders in a file system object.
