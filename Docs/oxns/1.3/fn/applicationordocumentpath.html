Functions

applicationOrDocumentPath

Supported By

  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Standard Module (1.0, 1.1, 1.2, 1.3, 1.4)

Syntax

[the] applicationOrDocumentPath of factor

applicationOrDocumentPath ( expression )

Factor and expression yield strings.

Description

The applicationOrDocumentPath function returns the path of the application or document with the specified name. If no such application or document can be found, the applicationOrDocumentPath function returns empty. In OpenXION, the locations searched for applications and documents are determined by the applicationPaths and documentPaths properties, respectively.

Note

The applicationOrDocumentPath function is specific to OpenXION; it is not part of the XION Scripting Language Standard.

Security

To use this function, OpenXION's security settings must allow the FILE_SYSTEM_READ security key. If the required security key is denied, a script error will be triggered.

Synonyms

appOrDocPath, path, progOrDocPath, programOrDocumentPath
