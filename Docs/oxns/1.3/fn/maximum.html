Functions

maximum

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Standard Module (1.0, 1.1, 1.2, 1.3, 1.4)
  • HyperTalk (2.2)

Syntax

[the] maximum of factor

maximum ( expression )

Factor and expression yield any kind of list.

Description

The maximum function returns the largest element of its arguments. The elements are compared as if by the > operator.

Synonym

max
