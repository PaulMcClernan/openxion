Functions

numToUni

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Standard Module (1.0, 1.1, 1.2, 1.3, 1.4)

Syntax

[the] numToUni of factor

numToUni ( expression )

Factor and expression yield non-negative integers.

Example

put numToUni(915) into it -- yields Î“

Description

The numToUni function returns the character whose Unicode code point is that of the integer passed to it.
