Functions

min

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Standard Module (1.0, 1.1, 1.2, 1.3, 1.4)
  • HyperTalk (2.2)

Syntax

[the] min of factor

min ( expression )

Factor and expression yield any kind of list.

Description

The min function returns the smallest element of its arguments. The elements are compared as if by the < operator.

Synonym

minimum
