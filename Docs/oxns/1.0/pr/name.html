Properties

name

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Standard Module (1.0, 1.1, 1.2, 1.3, 1.4)
  • HyperTalk (2.2)

Applies To

  • File system objects (disks, folders, files, and forks)
  • User-defined objects

Example

get the name of the first file

Description

The name property determines or changes the name of an object.
