Examples

read from file thePath until eof

read from file thePath until end

Description

A one-length string consisting solely of Unicode character U+FFFF, used to indicate when the read command has reached the end of a text file. Since U+FFFF is not a valid Unicode character, end will never appear anywhere else in a correctly-encoded text file.

Note

In HyperTalk, eof is actually a keyword to the read command, not a constant. We know this because we can put it into eof in HyperTalk, but we cannot put it into pi or any other real constant. However, the HyperCard Script Language Guide mistakenly lists eof as a constant. In XION, eof is a real constant.
