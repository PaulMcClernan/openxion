Constants

colon

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Standard Module (1.0, 1.1, 1.2, 1.3, 1.4)
  • HyperTalk (2.2)

Example

put "Macintosh HD" & colon & "Applications" & colon & "HyperCard" into thePath

Description

A one-length string consisting of a single ASCII colon character (:). This is used as the file path separator on Mac OS Classic systems.
