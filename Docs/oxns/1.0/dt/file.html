Data Types

file

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Standard Module (1.0, 1.1, 1.2, 1.3, 1.4)

Descriptors

  • index descriptor: put file 12 into steve
  • index descriptor: put file 12 of steve into bill
  • ordinal descriptor: put the fifth file into steve
  • ordinal descriptor: put the fifth file of steve into bill
  • name descriptor: put file "steve" into bill
  • name descriptor: put file "andy" of steve into bill

Properties

name, path, modificationDate, size

Description

The file data type represents a file in the file system. A file may contain forks.

Note

If no parent disk, volume, folder, or directory is specified, the current directory is used as the parent.

Security

To use files, OpenXION's security settings must allow the FILE_SYSTEM_READ security key. If the required security key is denied, a script error will be triggered.
