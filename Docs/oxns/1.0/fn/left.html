Functions

left

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Standard Module (1.0, 1.1, 1.2, 1.3, 1.4)

Syntax

[the] left of ( string , length )

left ( string , length )

String yields any string. Length yields an integer.

Description

The left function returns the leftmost length characters of the specified string.

Note

If length is zero or negative, empty is returned. If length is longer than the specified string, the original string is returned.
