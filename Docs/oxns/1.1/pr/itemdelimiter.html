Properties

itemDelimiter

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Standard Module (1.0, 1.1, 1.2, 1.3, 1.4)
  • HyperTalk (2.2)

Applies To

Global environment

Examples

if the itemDelimiter <> comma
then set the itemDelimiter to comma

set itemDelimiter to "#"

Description

The itemDelimiter property determines or changes the character that delimits item chunks. The default value is comma.

Changing the item delimiter has no effect on other data types, such as point or rectangle, or on functions, such as dateItems.
