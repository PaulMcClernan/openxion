Properties

modificationDate

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Standard Module (1.0, 1.1, 1.2, 1.3, 1.4)

Applies To

File system objects (disks, folders, files, and forks)

Example

get the modificationDate of the first file

Description

The modificationDate property determines or changes the modification date of a file system object.
