Other Keywords

__END__

Supported By

  • OpenXION (1.4)
  • OpenXION Base (1.4)

Syntax

__END__

Description

__END__ is a special keyword that tells the XION interpreter to stop parsing the XION script. Everything following a __END__ token will be ignored completely.

Compatibility

__END__ is not part of the XION Scripting Language Standard and is not portable. It is not available in versions of OpenXION before 1.4.
