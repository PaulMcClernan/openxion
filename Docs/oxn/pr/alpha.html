Properties

alpha

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.2, 1.3, 1.4)
  • OpenXION Standard Module (1.2, 1.3, 1.4)

Applies To

Colors

Example

get the alpha of purple

Description

The alpha property determines the alpha channel of a color. The alpha is specified as an integer between zero (transparent) and 65535 (opaque), inclusive.
