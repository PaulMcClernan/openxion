Functions

vmVersion

Supported By

  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Extended Module (1.0, 1.1, 1.2, 1.3, 1.4)

Syntax

the vmVersion

vmVersion ()

Example

put the vmVersion

Description

The vmVersion function returns the version of the Java virtual machine. You might use this to determine whether a script will run correctly under that version of the virtual machine.

Note

OpenXION gets the value of this function from the java.vm.version system property.

Security

To use this function, OpenXION's security settings must allow the SYSTEM_INFO security key. If the required security key is denied, a script error will be triggered.
