Properties

id

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Standard Module (1.0, 1.1, 1.2, 1.3, 1.4)
  • HyperTalk (2.2)

Applies To

User-defined objects

Example

get the id of steve

Description

The id property determines the unique ID number of an object. An object's ID number will never change during the object's lifetime and will be unique among objects of the same type.
