Functions

xionVersion

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Standard Module (1.0, 1.1, 1.2, 1.3, 1.4)

Syntax

the xionVersion

xionVersion ()

Example

put the xionVersion

Description

The xionVersion function returns the version of the XION interpreter. You might use this to determine whether a script will run correctly under that version of the interpreter.
