Operators

is fewer than

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Base (1.0, 1.1, 1.2, 1.3, 1.4)
  • HyperTalk (2.2)

Syntax

expression is fewer than expression

Description

The is fewer than operator compares the two given values. If the first value is less than the second, the is fewer than operator evaluates to true. If the first value is greater than or equal to the second, the is fewer than operator evaluates to false.

If both given values are integers or numbers, the values will be compared numerically. If both given values are dates, the values will be compared chronologically. Otherwise, the values will be compared as strings.

Note

String comparison is case-insensitive. For case-sensitive comparison, use the csstrcmp function.

Synonyms

comes before, is before, is less than, is smaller than, <
