External Languages

appleScript

Supported By

  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Extended Module (1.0, 1.1, 1.2, 1.3, 1.4)
  • HyperTalk (2.2)

Description

Executes the specified text as an AppleScript script.

Notes

This external language is only available when OpenXION is running under Mac OS X.

In OpenXION 1.2 and later, the output of the script, if any, can be retrieved using the result function. (OpenXION 1.0 and 1.1 did not capture any output.)
