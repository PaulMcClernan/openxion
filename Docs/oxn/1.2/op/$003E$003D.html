Operators

>=

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Base (1.0, 1.1, 1.2, 1.3, 1.4)
  • HyperTalk (2.2)

Syntax

expression >= expression

Description

The >= operator compares the two given values. If the first value is greater than or equal to the second, the >= operator evaluates to true. If the first value is less than the second, the >= operator evaluates to false.

If both given values are integers or numbers, the values will be compared numerically. If both given values are dates, the values will be compared chronologically. Otherwise, the values will be compared as strings.

Note

String comparison is case-insensitive. For case-sensitive comparison, use the csstrcmp function.

Synonyms

is not before, is not fewer than, is not less than, is not smaller than, isn't before, isn't fewer than, isn't less than, isn't smaller than, =>, ≥
