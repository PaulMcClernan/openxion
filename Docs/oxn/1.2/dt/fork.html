Data Types

fork

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Standard Module (1.0, 1.1, 1.2, 1.3, 1.4)

Descriptors

  • index descriptor: put fork 12 of steve into bill
  • ordinal descriptor: put the fifth fork of steve into bill
  • name descriptor: put fork "andy" of steve into bill

Properties

name, size

Description

The fork data type represents a fork, alternate stream, or metadata of a file in the file system.

Security

To use forks, OpenXION's security settings must allow the FILE_SYSTEM_READ security key. If the required security key is denied, a script error will be triggered.
