Properties

dialingTime

Supported By

  • OpenXION (1.2, 1.3, 1.4)
  • OpenXION Audio Module (1.2, 1.3, 1.4)
  • HyperTalk (2.2)

Applies To

Global environment

Description

The dialingTime property determines the amount of time the dial command waits before closing the communications port after issuing the modem command to dial a phone number. The amount of time is specified in ticks, or sixtieths of a second. The default value is 180 ticks, or 3 seconds.
