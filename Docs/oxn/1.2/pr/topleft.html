Properties

topLeft

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Standard Module (1.0, 1.1, 1.2, 1.3, 1.4)
  • HyperTalk (2.2)

Applies To

Rectangles

Example

get the topLeft of ((100,50,600,230) as rectangle)

Description

The topLeft property determines the top left corner of a rectangle.
