Properties

columnDelimiter

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Standard Module (1.0, 1.1, 1.2, 1.3, 1.4)

Applies To

Global environment

Example

set columnDelimiter to delete

Description

The columnDelimiter property determines or changes the character that delimits column chunks. The default value is Unicode character U+FFF0.

See Also

column, itemDelimiter, rowDelimiter
