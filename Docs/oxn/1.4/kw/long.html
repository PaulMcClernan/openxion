Other Keywords

long

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Base (1.0, 1.1, 1.2, 1.3, 1.4)
  • HyperTalk (2.2)

Applies To

Functions and properties

Syntax

[the] long function-or-property-name [of variant]

Example

get the long date

Description

The long keyword is a modifier that appears before a function call or property descriptor. It tells the function or property getter that the resulting value should be in a long form. The best example of this is the date function.
