Functions

heapSpace

Supported By

  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Extended Module (1.0, 1.1, 1.2, 1.3, 1.4)
  • HyperTalk (2.2)

Syntax

the heapSpace

heapSpace ()

Example

put the heapSpace into heapOFun

Description

In OpenXION, the heapSpace function returns the number of bytes of heap space currently available to the Java virtual machine.

In HyperTalk, the heapSpace function returns the number of bytes of heap space currently available to HyperCard.

Security

To use this function, OpenXION's security settings must allow the SYSTEM_INFO security key. If the required security key is denied, a script error will be triggered.
