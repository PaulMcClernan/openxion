Functions

serialPorts

Supported By

  • OpenXION (1.2, 1.3, 1.4)
  • OpenXION Audio Module (1.2, 1.3, 1.4)
  • HyperTalk (2.2)

Syntax

the serialPorts

serialPorts ()

Example

put the serialPorts

Description

The serialPorts function returns a lineEnding-delimited list of communications ports. These are the possible values of the dialingPort property, which determines the port used to dial phone numbers with a modem.

Note

In HyperTalk, this function existed but was undocumented.

Security

To use the serialPorts function, OpenXION's security settings must allow the SYSTEM_INFO security key. If the required security key is denied, a script error will be triggered.
