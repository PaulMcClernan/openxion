Functions

llength

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.2, 1.3, 1.4)
  • OpenXION Standard Module (1.2, 1.3, 1.4)

Syntax

[the] llength of factor

llength ( expression )

Factor and expression yield lists.

Description

The llength function returns the number of elements in a list.
