Ordinals

twelfth

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Standard Module (1.0, 1.1, 1.2, 1.3, 1.4)

Example

put twelfth char of "Hello, my name is Rebecca. But you can call me Beckie." -- prints "a" from "name"

Description

An ordinal corresponding to index 12.
