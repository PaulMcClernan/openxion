Data Types

object

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Standard Module (1.0, 1.1, 1.2, 1.3, 1.4)

Descriptors

  • index descriptor: put object 12 into steve
  • ordinal descriptor: put the fifth object into steve
  • ID descriptor: put object id 1719 into steve
  • name descriptor: put object "steve" into bill

Description

The object data type is the root of the class hierarchy of any user-defined data type.

Note

Converting an object to a string results in a string that looks like a descriptor. Conversely, a string that is formatted like a descriptor may be converted to the object it describes.
