Operators

is not precisely an element in

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Base (1.0, 1.1, 1.2, 1.3, 1.4)

Syntax

expression is not precisely an element in list

Description

The is not precisely an element in operator evaluates to true if none of the elements of the given list are the same type as and identical to the given expression. Otherwise, it evaluates to false.

Note

For a looser definition of equality (as specified for the = operator as opposed to the === operator), see is not an element of.

Synonyms

is not precisely an element of, isn't precisely an element in, isn't precisely an element of
