Operators

is precisely a

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Base (1.0, 1.1, 1.2, 1.3, 1.4)

Syntax

expression is precisely a type

Description

The is precisely a operator evaluates to true if the given value's data type is the given type. If the given value's data type is not the given type, the is precisely a operator evaluates to false. This is a stricter requirement than the one for the is an operator: the value must be compatible with the given type without requiring conversion (from a number to a string, for example).

Synonyms
