Operators

does not contain

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Base (1.0, 1.1, 1.2, 1.3, 1.4)

Syntax

expression does not contain expression

Description

The does not contain operator evaluates to true if the string to the left of the operator does not contain the string to the right of the operator as a substring. Otherwise, it evaluates to false.

Note

The does not contain operator is case-insensitive. For case-sensitive comparison, use the csoffset function.

Synonym

doesn't contain
