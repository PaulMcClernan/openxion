Operators

ref of

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.3, 1.4)
  • OpenXION Base (1.3, 1.4)

Syntax

ref of factor

Factor yields a reference.

Description

The ref of operator evaluates to the variant to which the given reference refers.

Synonyms

referent of, the ref of, the referent of

See Also

a reference to, reference
