Constants

gray

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Standard Module (1.0, 1.1, 1.2, 1.3, 1.4)

Example

set the color of myObject to gray

Description

A color with RGB values 32768, 32768, 32768. A medium gray color.

Note

Although HyperCard had no native support for color, the color constants originate from HyperTalk's visual command, which provided black, gray, grey, and white as possible images to transition to.

Synonym

grey
