Data Types

volume

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Standard Module (1.0, 1.1, 1.2, 1.3, 1.4)

Descriptors

  • index descriptor: put volume 12 into steve
  • ordinal descriptor: put the fifth volume into steve
  • name descriptor: put volume "steve" into bill

Properties

name, path, count

Description

The volume data type represents a mounted volume in the file system. A volume may contain folders and files.

Security

To use volumes, OpenXION's security settings must allow the FILE_SYSTEM_READ security key. If the required security key is denied, a script error will be triggered.

Synonym

disk
