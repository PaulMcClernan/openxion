Functions

version

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Standard Module (1.0, 1.1, 1.2, 1.3, 1.4)
  • HyperTalk (2.2)

Syntax

[the] version of factor

version ( expression )

Factor and expression yield some object or the name of some component that has a version number.

Description

The version function returns the version number of an object or component.

Security

To use this function, OpenXION's security settings must allow the SYSTEM_INFO security key. If the required security key is denied, a script error will be triggered.
