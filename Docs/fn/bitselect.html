Functions

bitSelect

Supported By

  • OpenXION (1.4)

Syntax

[the] bitSelect of ( expression , expression )

bitSelect ( expression , expression )

Both expressions yield integers.

Description

The bitSelect function takes the bits from the first given integer corresponding to the set bits in the second integer and packs them to the right. For example, bitSelect(179,201) (10110011 and 11001001 in binary) will produce 9 (1001 in binary).

If both given values are negative, the result will be negative. Otherwise, the result will be positive.
