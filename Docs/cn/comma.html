Constants

comma

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Standard Module (1.0, 1.1, 1.2, 1.3, 1.4)
  • HyperTalk (2.2)

Example

set the itemDelimiter to comma

Description

A one-length string consisting of a single ASCII comma character (,). This is used as the default itemDelimiter, as well as the delimiter when converting a list to a string.
