Data Types

URLs

Supported By

  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Extended Module (1.0, 1.1, 1.2, 1.3, 1.4)

Description

The URLs data type represents a list that consists entirely of URLs.

Note

Opening a URL with the open command without specifying an I/O method opens the URL in the user's default browser. Opening a URL while specifying an I/O method allows using the read and write commands to read from and write to the URL.
