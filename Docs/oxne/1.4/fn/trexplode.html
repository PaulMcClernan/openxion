Functions

trExplode

Supported By

  • OpenXION (1.3, 1.4)
  • OpenXION Extended Module (1.3, 1.4)

Syntax

[the] trExplode of ( string , delimiter )

trExplode ( string , delimiter )

String yields any string. Delimiter yields a string describing a set of characters.

Description

The trExplode function returns a list of the chunks in the specified string delimited by any of the characters in the specified delimiter, not including any trailing empty chunks.

Delimiter describes a set of characters, not a string. A range of characters can be specified by putting a hyphen between two characters; to match a literal hyphen, put a hyphen at the beginning or end. For example, the following pattern matches all numbers and basic Latin letters, as well as hyphens: A-Za-z0-9-
