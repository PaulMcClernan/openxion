Data Types

files

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Standard Module (1.0, 1.1, 1.2, 1.3, 1.4)

Descriptors

  • index range descriptor: put files 2 through 7 into steve
  • index range descriptor: put files 2 through 7 of steve into bill
  • ordinal range descriptor: put the third through eighth files into steve
  • ordinal range descriptor: put the third through eighth files of steve into bill
  • mass descriptor: put the files into steve
  • mass descriptor: put the files of steve into bill

Description

The files data type represents a list that consists entirely of files.

Note

If no parent disk, volume, folder, or directory is specified, the current directory is used as the parent.

Security

To use files, OpenXION's security settings must allow the FILE_SYSTEM_READ security key. If the required security key is denied, a script error will be triggered.
