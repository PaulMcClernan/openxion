Functions

variance

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Standard Module (1.0, 1.1, 1.2, 1.3, 1.4)

Syntax

[the] variance of factor

variance ( expression )

Factor and expression yield lists of integers, numbers, or complexes.

Example

get variance(1,2,3,4)

Description

The variance function returns the population variance of a list of numeric values passed to it.

If all the values are integers or numbers, a number is returned. If any value is a complex, a complex is returned.

Synonym

pvariance

See Also

svariance, stddev
