Properties

green

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Standard Module (1.0, 1.1, 1.2, 1.3, 1.4)

Applies To

Colors

Example

get the green of purple

Description

The green property determines the intensity of the green channel of a color. The intensity is specified as an integer between zero and 65535, inclusive.
