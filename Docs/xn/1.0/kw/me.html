Other Keywords

me

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Base (1.0, 1.1, 1.2, 1.3, 1.4)
  • HyperTalk (2.2)

Syntax

me

Example

get the short name of me

Description

Me is a special keyword that always evaluates to the object containing the currently running handler. If there is no such object, me evaluates to the interpreter instead.
