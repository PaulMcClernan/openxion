Data Types

binary

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Standard Module (1.0, 1.1, 1.2, 1.3, 1.4)

Description

The binary data type represents a stream of bytes. In string form, a binary appears as a string of hexadecimal digits.

Notes

A binary is manipulated using the chunk types byte, tinyInt, shortInt, medInt, longInt, halfFloat, singleFloat, and doubleFloat.

Two global properties, unsigned and littleEndian, control the signedness and byte order of the binary chunk types, respectively. The unsigned property is false, or signed, by default. The littleEndian property is false, or big-endian, by default.
