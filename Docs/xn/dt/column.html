Data Types

column

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Standard Module (1.0, 1.1, 1.2, 1.3, 1.4)

Descriptors

  • index descriptor: put column 12 of steve into bill
  • index range descriptor: put column 2 through 7 of steve into bill
  • ordinal descriptor: put the fifth column of steve into bill
  • ordinal range descriptor: put the third through eighth column of steve into bill
  • name descriptor: put column "andy" of steve into bill

Description

The column data type is a chunk type of string that represents a chunk of text delimited by the columnDelimiter. This includes empty columns but not the last trailing empty column of the string, if there is one.

Note

Accessing a column chunk that is out of range returns empty. Modifying a column chunk that is out of range inserts as many columnDelimiters as necessary to insert text in the appropriate column of the parent string.

Synonym

col
