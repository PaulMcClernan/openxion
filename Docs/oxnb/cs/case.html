Supported By

  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Base (1.0, 1.1, 1.2, 1.3, 1.4)

Syntax

switch expression
case expression

Description

The case keyword is part of the switch control structure. See the description for switch for details.
