Control Structures

catch

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Base (1.0, 1.1, 1.2, 1.3, 1.4)

Syntax

try
   statementList
catch variable
   statementList
end try

Description

The catch keyword is part of the try control structure. See the description for try for details.
