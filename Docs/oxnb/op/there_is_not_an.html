Operators

there is not an

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Base (1.0, 1.1, 1.2, 1.3, 1.4)
  • HyperTalk (2.2)

Syntax

there is not an factor

Factor is a descriptor.

Description

The there is not an operator evaluates to true if no variant with the given descriptor exists, or false if a variant with the given descriptor does exist.

Synonyms

there is no, there is not a, there isn't a, there isn't an, there's no

See Also
