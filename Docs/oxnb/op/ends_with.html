Operators

ends with

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Base (1.0, 1.1, 1.2, 1.3, 1.4)

Syntax

expression ends with expression

Description

The ends with operator evaluates to true if the string to the left of the operator ends with the string to the right of the operator. Otherwise, it evaluates to false.

Note

The ends with operator is case-insensitive. For case-sensitive comparison, use the csoffset function.
