Operators

is equal to

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Base (1.0, 1.1, 1.2, 1.3, 1.4)
  • HyperTalk (2.2)

Syntax

expression is equal to expression

Description

The is equal to operator evaluates to true if the given values are equal. Otherwise, it evaluates to false.

If both given values are integers or numbers, the values will be compared numerically. If both given values are dates, the values will be compared chronologically. Otherwise, the values will be compared as strings.

Note

String comparison with the is equal to operator is case-insensitive. For case-sensitive comparison, use the csstrcmp function or the === operator.

Synonyms

equals, is, =, ==

See Also

===, eq
