Operators

gt

Supported By

  • OpenXION (1.0, 1.1, 1.2, 1.3)
  • OpenXION Base (1.0, 1.1, 1.2, 1.3)

Syntax

expression gt expression

Description

The gt operator

The gt operator compares the two given values as strings. If the first string is greater than the second, the gt operator evaluates to true. If the first string is less than or equal to the second, the gt operator evaluates to false.

Note

String comparison is case-insensitive. For case-sensitive comparison, use the csstrcmp function.

Compatibility

The gt operator is no longer available in OpenXION 1.4 and later.
