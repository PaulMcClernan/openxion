Operators

is in

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Base (1.0, 1.1, 1.2, 1.3, 1.4)
  • HyperTalk (2.2)

Syntax

expression is in expression

Description

The is in operator evaluates to true if the string to the right of the operator contains the string to the left of the operator as a substring. Otherwise, it evaluates to false.

Note

The is in operator is case-insensitive. For case-sensitive comparison, use the csoffset function.

Synonym

is of
