Operators

there's an

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Base (1.0, 1.1, 1.2, 1.3, 1.4)
  • HyperTalk (2.2)

Syntax

there's an factor

Factor is a descriptor.

Description

The there's an operator evaluates to true if a variant with the given descriptor exists, or false if no variant with the given descriptor exists.

Synonyms

there is a, there is an, there's a

See Also

there is not a, there is not an
