Supported By

  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Base (1.0, 1.1, 1.2, 1.3, 1.4)
  • HyperTalk (2.2)

Description

Executes the specified text as a script. The string is executed as though it were inserted in place of the do statement.

Compatibility

In OpenXION, the hyperTalk "language" is provided as a synonym of xion and openxion for compatibility purposes.
