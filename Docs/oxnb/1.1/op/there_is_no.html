Operators

there is no

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Base (1.0, 1.1, 1.2, 1.3, 1.4)
  • HyperTalk (2.2)

Syntax

there is no factor

Factor is a descriptor.

Description

The there is no operator evaluates to true if no variant with the given descriptor exists, or false if a variant with the given descriptor does exist.

Synonyms

there is not a, there is not an, there isn't a, there isn't an, there's no
