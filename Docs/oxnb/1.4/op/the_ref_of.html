Operators

the ref of

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.3, 1.4)
  • OpenXION Base (1.3, 1.4)

Syntax

the ref of factor

Factor yields a reference.

Description

The the ref of operator evaluates to the variant to which the given reference refers.

Synonyms

ref of, referent of, the referent of
