Operators

===

Supported By

  • XION Scripting Language Standard (1.0)
  • OpenXION (1.0, 1.1, 1.2, 1.3, 1.4)
  • OpenXION Base (1.0, 1.1, 1.2, 1.3, 1.4)

Syntax

expression === expression

Description

The === operator evaluates to true if the given values have the same type and are identical. Otherwise, it evaluates to false.

Note

String comparison with the === operator is case-sensitive. For case-insensitive comparison, use the = or eq operator.

Synonym

is exactly

See Also

=, eq, equal
