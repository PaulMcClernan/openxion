Functions

voices

Supported By

  • OpenXION (1.2, 1.3, 1.4)
  • OpenXION Audio Module (1.2, 1.3, 1.4)
  • HyperTalk (2.2)

Syntax

the voices

voices ()

Example

put the voices

Description

The voices function returns a lineEnding-delimited list of voices currently available for use with the speak command.
