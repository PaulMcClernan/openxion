Functions

sounds

Supported By

  • OpenXION (1.2, 1.3, 1.4)
  • OpenXION Audio Module (1.2, 1.3, 1.4)

Syntax

the sounds

sounds ()

Example

put the sounds

Description

The sounds function returns a lineEnding-delimited list of instrument names currently available for use with the play command.
