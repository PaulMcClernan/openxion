Properties

dialingPort

Supported By

  • OpenXION (1.2, 1.3, 1.4)
  • OpenXION Audio Module (1.2, 1.3, 1.4)
  • HyperTalk (2.2)

Applies To

Global environment

Description

The dialingPort property determines which communications port is used by the dial command to dial a phone number using a modem. A list of possible values for this property is returned by the serialPorts function.

Security

To use the dialingPort property, OpenXION's security settings must allow the TELEPHONY security key. If the required security key is denied, a script error will be triggered.

Compatibility

In HyperTalk, this property existed but was undocumented.
