Properties

toneChannel

Supported By

  • OpenXION (1.2, 1.3, 1.4)
  • OpenXION Audio Module (1.2, 1.3, 1.4)

Applies To

Global environment

Description

The toneChannel property determines which channel tones produced by the tone command is played on. Only one tone can be played on a single channel at any one time, but multiple tones can be played simultaneously on different channels. OpenXION supports 8 channels, numbered 1 through 8.

Note

Music played with the play command and tones produced by the tone command are always played on separate channels, despite the similarities between the soundChannel and toneChannel properties.
